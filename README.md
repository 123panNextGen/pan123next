# Pan123 Next

_**随叫随用**_

这是一个使用 `Dart/Flutter` 编写的第三方 123盘 客户端,
 采用 **Fluent UI** 样式,
 并且支持许多简单且易用的功能

## 使用

首先, 确保你已准备以下环境:

|名称|版本|
|---|---|
|Flutter SDK|Any / ~Latest|

随后克隆项目并安装依赖

```bash
git clone https://github.com/123panNextGen/pan123next.git
cd pan123next

flutter pub get # 获取依赖
```

一切准备就绪, 就可以运行了

```bash
flutter run
```

要想修改源码, 可在 `lib/` 中翻阅
